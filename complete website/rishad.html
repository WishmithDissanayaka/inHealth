Gallery page

Our gallery page is designed to captivate users with interactive thumbnails that signal their clickable nature upon hover. Clicking on a thumbnail reveals an extended view featuring a larger image and detailed description Each image and description is equipped with appropriate alternative text for accessibility. Additionally, users can personalize their viewing experience by customizing page color and font style within the extended view, enhancing engagement and connection with our content.

Sitemap map

Our website's sitemap is developed using Scalable Vector Graphics (SVG), offering an interactive and accessible representation of its hierarchical structure. Each page is depicted as a node within the SVG canvas, featuring hover effects and clickable links for seamless navigation. The sitemap is seamlessly integrated into the website's design and adjusts responsively to different screen sizes. Accessibility is prioritized with alt text for each node, and keyboard navigation is supported for enhanced usability.
